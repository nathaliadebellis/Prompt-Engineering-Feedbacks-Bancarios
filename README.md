# 💬 Extraindo Insights do Feedback de Clientes Bancários com IA

## 📖 Sobre o Projeto

Este projeto foi desenvolvido durante o **Bootcamp Bradesco – GenAI, Dados & Cyber**, promovido pela DIO.

O objetivo foi aplicar conceitos de **Prompt Engineering** para construir um prompt estruturado capaz de orientar um modelo de Inteligência Artificial na análise de feedbacks de clientes bancários.

O prompt foi elaborado seguindo boas práticas de Engenharia de Prompt, incluindo definição de contexto, objetivo, critérios de análise, formato esperado da resposta e restrições, permitindo que a IA produza análises claras, organizadas e úteis para apoiar a tomada de decisão.

---

# 🎯 Objetivos

- Aplicar técnicas de Prompt Engineering.
- Construir prompts claros e objetivos.
- Definir contexto, papel da IA e formato esperado da resposta.
- Extrair insights relevantes a partir de feedbacks de clientes.
- Demonstrar boas práticas no uso de IA Generativa.

---

# 🧠 Cenário

O prompt foi desenvolvido para analisar comentários de clientes sobre serviços bancários, como:

- Aplicativo do banco
- PIX
- Cartão de crédito
- Internet Banking
- Atendimento digital

A análise busca identificar padrões, problemas recorrentes, oportunidades de melhoria e pontos positivos da experiência dos clientes.

# 📈 Aprendizados

Durante o desenvolvimento deste desafio foi possível compreender como a qualidade das respostas geradas por uma IA depende diretamente da qualidade do prompt elaborado.

Também foi possível aplicar conceitos como:

- definição clara do objetivo;
- contextualização da tarefa;
- delimitação das restrições;
- definição do formato esperado da resposta;
- construção de instruções específicas para orientar o modelo.

Essas práticas tornam as respostas mais consistentes, confiáveis e alinhadas às necessidades do usuário.
